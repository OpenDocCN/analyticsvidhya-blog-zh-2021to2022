一年级和二年级学生的明确路线图！

嗨伙计们！这篇博客是关于如果我必须重新开始，我会如何学习数据科学和相关领域的。如果您是一年级或二年级学生，并且希望深入了解任何科技公司的数据应用程序的端到端设计，以便在该领域拥有光明的职业生涯，请遵循本指南！我保证，如果你沿着这条路走下去，你将拥有在这个行业工作至少3年的人的知识。

免责声明-

  1. 这不是免费的指南。据我估计，遵循整个路线图将花费你近60，000印度卢比(不包括消费税)。然而，相信我，这项投资会比教练机构的所谓认证培训好得多，后者会花费你同样多的钱，如果不是更多的话。
  2. 这并不能保证你一毕业就能在LPA找到一份30-40英镑的工作。我只能保证深入的知识，但不能保证从这门课程中获得这样的薪水。如果你想这样，请随意走竞争性编程和开源之路。
  3. 我与这两门课程都没有任何关系。我只是一名学生，既不认识这些机构，也不认识它们的创始人。🙂

如果您仍然感兴趣，请继续阅读，并按照路线图的顺序进行操作。

Python-

我想指出三个资源。根据您的时间限制，您可以使用任意或全部-

  1. Python for Everybody-Charles Severance on Coursera-坦白地说，这是60-70%的Python开发者的第一门课程。你将会被密歇根大学一位非常酷的教授教授。
  2. Python playlist by DHA val Patel-你将从一位与英伟达和彭博合作过的软件工程师那里学到东西。如果你想进入这个领域，也可以考虑跟随他的频道。
  3. Python播放列表作者Tim- 年轻的软件工程师，你们大多数人都能与之产生共鸣。他有3个播放列表-初级，中级和高级。你也可以考虑从他的渠道构建一些很酷的Python项目。值得订阅！

页（page的缩写）如果你有时间，可以考虑看看Tim或Charles Severance教授的Flask/Django框架(理论性更强)。但是你现在可以跳过它。

SQL-

从 freeCodeCamp 或 Mosh 了解这一点。这里最关键的是练习。从Leetcode、Hackerrank和SQLZoo中完成。

数据结构和算法-

听起来可能有悖直觉，但我建议不要用Python来做这件事。选择C++或Java，每天练习1-2 leet code easy to medium，以防你不得不回到传统的软件工程工作。相信我，你会在YouTube上找到大量相关资源。😂

数据工程-

是的，你没听错！如果你从数据工程开始，你会有更好的数据科学和ML基础。在这个领域，TrendyTech是一门很好的课程。说实话，在这个领域没有很多高质量的实用课程。

注-数据工程是一个蓬勃发展的领域。如果你喜欢这门课程，你的旅程就到此为止了(至少要到大学)，没有必要继续读下去。完成课程，尽可能多地实习。熟练的数据工程师的工资与数据科学家一样高，如果不是更高的话，而且这个领域的熟练专业人员非常少，但需求却在不断增长。此外，这个领域比机器学习更受只有学士学位的新生欢迎。所以这条路很美好。

数据科学和机器学习-

现在，这是帖子最有趣的部分。老实说，应用课程无疑是这一领域最好的课程。导师斯里坎特·维尔马爵士(拥有在亚马逊和雅虎担任机器学习角色的丰富经验)是我见过的印度最好的ML导师，他的教学技能与斯坦福、麻省理工等大学的导师不相上下(如果不是，有时更好，XD)。他会用他讲课的理论性和实践性的完美融合，让你爱上机器学习。

注意-

  1. 这门课有30个作业，都很难。祝你好运。
  2. 除了课程讲座，你还会有一些现场会议。你可以忽略那些有经验的人，比如ML系统设计等。就目前而言。会有一些代码演练和面试问题，比如会话，你应该看看。
  3. 本课程结束时，你将拥有一个很好的项目组合。考虑使用Srikanth Sir在现场视频中讨论的方法(Flask、Streamlit、Sagemaker、KubeFlow等)部署项目。这对我帮助很大。
  4. iNeuron也有数据科学课程。它更侧重于编码应用，而不是理论。这是你的另一个选择。
  5. 这门课程涵盖了现实世界中70%的数据科学。其余的，请继续阅读。

实用数据科学-

尝试完成来自 LogikBot 的许多课程，以获得在现实世界中实际进行的数据科学的经验。阅读杰森·布朗利的电子书也是一种很棒的方式。基本了解编码方面。

结论-

哇哦。这需要很多时间。对于一个相对聪明的学生，我估计大约两年。如果你仍然有时间，考虑学习Flask/Django，观看AppliedAI的ML系统设计视频，特别是如果你的目标是机器学习工程师角色。
