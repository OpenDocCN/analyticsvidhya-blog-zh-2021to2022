在虚拟机(VMware)上安装Red Hat Enterprise Linux

原文：https://medium.com/analytics-vidhya/installation-of-red-hat-enterprise-linux-on-virtual-machine-vmware-372339fab02d?source=collection_archive---------14-----------------------#2021-08-16

简介:

Red Hat Enterprise Linux (RHEL)是为商业市场开发的Linux操作系统的发行版。RHEL以前被称为红帽Linux高级服务器。

RHEL操作系统(OS)支持物理、虚拟化和云环境中的各种工作负载。RHEL版本适用于服务器、大型机、SAP应用、台式机和OpenStack。

让我们开始安装:

我们必须做三件基本事情:

  1. 创建Linux虚拟机。
  2. 设备和DNS配置。
  3. YUM配置。
  • 首先安装VMware workstation，然后单击创建新虚拟机选项。
  • 然后选择典型选项，然后点击下一步。
  • 现在选择第二个选项“已安装的光盘镜像文件”,然后浏览linux ISO文件，然后单击下一步。
  • 提供您将在用户和root帐户中使用的用户名和密码。
  • 现在输入虚拟机名称，它将自动使用虚拟机的默认路径。
  • 设置磁盘空间，选择第一个选项将虚拟磁盘存储在单个文件上，然后单击下一个。
  • 现在点击完成
  • 现在您的虚拟机正在安装。
  • 现在选择‘其他’。
  • 输入root用户名，选择登录，然后输入您之前在安装时输入的密码
  • 现在右击虚拟机窗口并选择终端，现在终端窗口打开。使用setup命令并按enter键。
  • 现在，选择防火墙配置回车。
  • 进入启用选项，按空格键，然后选择确定。
  • 点击是按钮。
  • 选择网络配置并选择运行工具。
  • 选择设备配置，按保存并退出按钮。
  • 选择下图所示的第一个选项。
  • 写下主机名、主DNS、辅助DNS，然后点击确定按钮。重新启动虚拟机..
  • 提供静态IP，该IP就是您的虚拟机IP。设置默认网关，然后按“确定”按钮。
  • 进入虚拟机— >点击设置— >选择CD/DVD(SDATA) — >选择“用作ISO镜像”选项，然后浏览RHEL的ISO镜像。在“设备状态”中，勾选“已连接”和“开机时已连接”复选框。
  • 现在一个RHEL文件夹将会打开。
  • 打开终端，在终端上键入以下所有命令。
cd /var/

通过这个命令，您可以到达var目录。

ls

命令列出所有目录。

mkdir

做一个目录。

cd /var/ftp

进入ftp。

  • 创建一个目录名“pub”。
  • 现在“pub”是一个空目录，所以使用下面的命令复制pub目录中的“Packages”文件夹，并按回车键，然后在pub中复制“Package”文件夹。
“cp -R Packages /var/ftp/pub”
  • 现在进入包文件夹，运行下面终端图像中显示的命令。
  • 现在运行命令来配置yum。使用命令如下图所示。
  • 现在从使用命令返回到“pub”返回一步。
cd ..
  • 现在运行以下命令。
“createrepo -v Packages”
  • 运行命令到达yum.repos.d位置。
cd /etc/yum.repos.d
  • 现在编辑路径并给出所有信息，如下所示。
press Esc ->colon(:)->wq!(for save and exit)
  • 再次键入命令进行检查。
vi testing.repo
Type Command for cleaning
  • 键入清洗命令。
yum clean all
  • 键入安装java的命令。
yum install java

安装完成。
