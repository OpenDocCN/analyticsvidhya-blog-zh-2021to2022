对于任何机器学习问题，我们的第一步都是选择合适的算法。为了测试使用哪种算法，我们必须绘制数据并查看变量之间的关系。

在线性回归模型中，我们基本上要做的是尝试找到最佳拟合线**以最小误差拟合数据中的特征。**

当我们决定好算法的类型后，现在我们将继续分配因变量和自变量。

我用 X 和 Y 来表示变量。工作年限是一个独立变量，工资取决于它，因为工资会随着工作年限的增减而增减。
