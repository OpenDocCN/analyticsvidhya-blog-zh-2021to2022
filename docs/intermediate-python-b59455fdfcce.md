## Python 中的数据类型

## 命名规格

在命名标识符时，Python 有一些命名约定。在用标识符命名变量、函数、类、模块或其他对象时，应该遵守这些规则。标识符应该总是以字母 ***(A-Z / a-z)*** 或下划线 ***(_)*** 开头，后跟零个或多个字母、下划线和数字 ***(0-9)*** 。Python 不允许标识符中包含任何其他字符，如 ***%、@、(*** )。Python 是区分大小写的。因此，单词 ***【你好】*** 和 ***【你好】*** 在 Python 中被认为是两个不同的单词。
