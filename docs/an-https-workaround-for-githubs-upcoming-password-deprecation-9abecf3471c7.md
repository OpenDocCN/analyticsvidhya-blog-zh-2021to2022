TL；速度三角形定位法(dead reckoning)

*   从 2021 年 8 月 13 日开始，Github 在认证 Git 操作时将不再接受账户密码
*   Github 将需要基于令牌或 ssh 密钥的认证
*   如果 SSH-keys 给你带来问题并且不起作用，你可以创建一个个人认证令牌并把它保存在你的配置文件中…
