# Python-

我想指出三个资源。根据您的时间限制，您可以使用任意或全部-

1.  [**Python for Everybody-Charles Severance on Coursera-**](https://www.coursera.org/specializations/python)坦白地说，这是 60-70%的 Python 开发者的第一门课程。你将会被密歇根大学一位非常酷的教授教授。
2.  [**Python playlist by DHA val Patel-**](https://www.youtube.com/playlist?list=PLeo1K3hjS3uv5U-Lmlnucd7gqF-3ehIh0)**你将从一位与英伟达和彭博合作过的软件工程师那里学到东西。如果你想进入这个领域，也可以考虑跟随他的频道。**
3.  **[**Python 播放列表作者 Tim-**](https://www.youtube.com/c/TechWithTim) 年轻的软件工程师，你们大多数人都能与之产生共鸣。他有 3 个播放列表-初级，中级和高级。你也可以考虑从他的渠道构建一些很酷的 Python 项目。值得订阅！**

**页（page 的缩写）如果你有时间，可以考虑看看 Tim 或 Charles Severance 教授的 Flask/Django 框架(理论性更强)。但是你现在可以跳过它。**

# **SQL-**

**从 [freeCodeCamp](https://www.youtube.com/watch?v=HXV3zeQKqGY) 或 [Mosh](https://www.youtube.com/watch?v=7S_tz1z_5bA) 了解这一点。这里最关键的是练习。从 Leetcode、Hackerrank 和 SQLZoo 中完成。**

# ****数据结构和算法-****

**听起来可能有悖直觉，但我建议不要用 Python 来做这件事。选择 C++或 Java，每天练习 1-2 leet code easy to medium，以防你不得不回到传统的软件工程工作。相信我，你会在 YouTube 上找到大量相关资源。😂**

# **数据工程-**

**是的，你没听错！如果你从数据工程开始，你会有更好的数据科学和 ML 基础。在这个领域，TrendyTech 是一门很好的课程。说实话，在这个领域没有很多高质量的实用课程。**

**注-数据工程是一个蓬勃发展的领域。如果你喜欢这门课程，你的旅程就到此为止了(至少要到大学)，没有必要继续读下去。完成课程，尽可能多地实习。熟练的数据工程师的工资与数据科学家一样高，如果不是更高的话，而且这个领域的熟练专业人员非常少，但需求却在不断增长。此外，这个领域比机器学习更受只有学士学位的新生欢迎。所以这条路很美好。**

# **数据科学和机器学习-**

**现在，这是帖子最有趣的部分。老实说，应用课程无疑是这一领域最好的课程。导师斯里坎特·维尔马爵士(拥有在亚马逊和雅虎担任机器学习角色的丰富经验)是我见过的印度最好的 ML 导师，他的教学技能与斯坦福、麻省理工等大学的导师不相上下(如果不是，有时更好，XD)。他会用他讲课的理论性和实践性的完美融合，让你爱上机器学习。**

**注意-**

1.  **这门课有 30 个作业，都很难。祝你好运。**
2.  **除了课程讲座，你还会有一些现场会议。你可以忽略那些有经验的人，比如 ML 系统设计等。就目前而言。会有一些代码演练和面试问题，比如会话，你应该看看。**
3.  **本课程结束时，你将拥有一个很好的项目组合。考虑使用 Srikanth Sir 在现场视频中讨论的方法(Flask、Streamlit、Sagemaker、KubeFlow 等)部署项目。这对我帮助很大。**
4.  **iNeuron 也有数据科学课程。它更侧重于编码应用，而不是理论。这是你的另一个选择。**
5.  **这门课程涵盖了现实世界中 70%的数据科学。其余的，请继续阅读。**

# ****实用数据科学-****

**尝试完成来自 [LogikBot](https://www.logikbot.com/) 的许多课程，以获得在现实世界中实际进行的数据科学的经验。阅读[杰森·布朗利](https://machinelearningmastery.com/products/)的电子书也是一种很棒的方式。基本了解编码方面。**
