作为一名软件工程师，我每天接触的工具之一是 Git。Git 是一个版本控制系统(VCS)。VCS 就是一个跟踪开发人员代码变化的软件，可以根据这些变化来回移动。外面有很多 VCS，但是最受开发者欢迎的是 Git。
