Numpy是Numerical Python的缩写，是许多其他数据科学库(如Pandas和Scikit-learn)所依赖的基础库。除了数据科学应用之外，numpy还有助于显著减少您的计算时间，并使您的代码更加高效。本教程将向您介绍Numpy的一些独特属性，尤其是ndarray。

设置

为了使用numpy，标准的导入方式如下:

import numpy as np

如果您的计算机上尚未安装numpy，请使用以下命令进行安装:

pip3 install numpy
