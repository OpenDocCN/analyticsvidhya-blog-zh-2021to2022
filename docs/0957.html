在Google Colab中渲染开放的人工智能健身房环境

原文：https://medium.com/analytics-vidhya/rendering-openai-gym-environments-in-google-colab-9df4e7d6f99f?source=collection_archive---------1-----------------------#2021-02-08

我已经在 Google Colab 中发布了一个用于渲染你的gym环境的模块。因为Colab运行在VM实例上，而VM实例不包含任何类型的显示器，所以在笔记本中进行渲染是很困难的。在浏览了各种方法之后，我发现使用moviepy库最适合在Colab中呈现视频。因此，我为此构建了一个包装类，名为colabgymrender。

装置

apt-get install -y xvfb python-opengl ffmpeg > /dev/null 2>&1
pip install -U colabgymrender

使用

在Recorder对象中包装一个健身房环境。

env = gym.make("CartPole-v0")
env = Recorder(env, <directory>, <fps>)

如果您通过指定帧速率，发布到的视频将采用该帧速率。否则，环境将检查由环境本身在env.metadata['video.frames_per_second']中指定的默认帧速率。如果都没有找到，帧速率将默认为30。

您可以使用env.pause()和env.resume()暂停或恢复录制，但请确保在调用env.play()之前env.resume()至少录制一帧。否则，当你试图播放一个没有记录任何帧的视频时，你会得到一个错误。

录制时，每次环境达到终端状态，视频会自动发布到。不需要手动释放录像。这提供了易用性，而不必手动控制记录器的操作。只需录制一集(或一集的一部分)，然后播放内容。
