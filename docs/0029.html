matplotlib:python的绘图库

atplotlib是一个全面的库，用于在Python中创建静态、动画和交互式可视化。都说“Matplotlib让容易的事情变得容易，让困难的事情变得可能”。这是一个低级的库，它的接口类似于Matlab，提供了很多自由，但代价是必须编写更多的代码。

在你的系统中安装Matplotlib。

pip install matplotlib
