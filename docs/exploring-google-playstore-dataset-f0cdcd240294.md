# 极端值

> 在统计学中，异常值是与其他观察值显著不同的数据点。

![](img/63e1ed291521f197a2c192f636355a1d.png)

大小的箱线图(mb)

这个应用程序大小的方框图显示，大多数应用程序的大小都在 75mb 以下。我们还可以找到一些 80 到 100mb 以上的异常值。
