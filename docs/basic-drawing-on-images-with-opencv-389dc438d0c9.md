## 如何使用 OpenCV 在图像上绘制线条、形状和自定义绘图。

可以使用 **OpenCV** 绘图功能在图像上绘制线条、常见形状和自定义形状。图像的坐标点和空间尺寸用于指定图像上的绘图的位置和尺寸。
