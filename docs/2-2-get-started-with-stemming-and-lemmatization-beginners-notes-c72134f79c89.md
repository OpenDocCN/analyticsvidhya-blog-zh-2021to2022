# 堵塞物

词干是自然语言处理中使用的一种文本规范化技术。在这个过程中，屈折词被转换成词干。这通常包括去掉单词中的任何词缀。由于这是一种基于规则的技术，被剥离的单词可能不总是有效的单词。
