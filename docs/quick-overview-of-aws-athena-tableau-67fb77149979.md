**AWS 雅典娜**

*   Athena 是无服务器的，不需要管理基础架构。
*   它的交互式查询服务使使用标准 SQL 分析数据变得容易。
*   没有 ETL——围绕 S3 桶创建一个外部包装表，并开始使用它。
