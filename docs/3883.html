SQL语法介绍

SELECT *
FROM <table>

下面的操作意味着选择表中的所有列。

* 是一个通配符运算符，用于返回给定表格中的所有行。

我们使用PostgreSQL作为我们的例子，但是MySQL有类似的命令，并且只是一个Google away，或者你可以使用这个指南
